You need to do two special things to the Id column. First, you need to mark the Id column  
as a primary key column by selecting the column in the Table Designer and clicking the icon  
of a key. LINQ to SQL requires you to specify your primary key columns when performing  
inserts or updates against the database.  
Next, you need to mark the Id column as an Identity column by assigning the value Yes to  
the
Is Identity
property (see Figure 3). An Identity column is a column that is assigned a  
new number automatically whenever you add a new row of data to a table.  
After you make these changes, save the table with the name tblMovie. You can save the  
table by clicking the
Save
button.
