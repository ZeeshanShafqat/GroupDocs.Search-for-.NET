After you create the new database, you can open the database by double-clicking the  
MoviesDB.mdf file in the App_Data folder. Double-clicking the MoviesDB.mdf file opens the  
Server Explorer window (see Figure 2).  
The Server Explorer window is called the Database Explorer  
window when using Visual Web Developer.  
Figure 2 – Using the Server Explorer window  
We need to add one table to our database that represents our movies. Right-click the Tables  
folder and select the menu option
Add New Table
. Selecting this menu option opens the  
Table Designer (see Figure 3).  
Figure 3 – The Table Designer
